# SnapLog 日報フォーマット

このドキュメントは、SnapLogが生成する日報Markdownの**固定フォーマット**を定義します。
日報本文はLLMが生成し、ヘッダーは `src/report.py` で付与されます。

## 出力構成（固定）

```markdown
# YYYY-MM-DD 日報

## 目的
（1〜2行で、今日の目的や狙い）

## やったこと
- 目的に対して実施した内容と結果

## 学び・気づき
- 技術的な発見やハマった点と解決策

## AI出力ログ
- 有用だったAIの出力（プロンプトと回答の要約、生成コードなど）
（該当なしの場合は「（なし）」と記載）

## 次やること
- 次回のTODOや未解決事項
```

## ルール

- 見出し順は固定。追加セクションは禁止。
- LLMの出力は**本文のみ**（`#` タイトルは出力しない）。
- 文章は簡潔に。箇条書きを基本とする。

## 関連実装

- テンプレート: `src/llm_client.py`
- ヘッダー付与/結合: `src/report.py`
